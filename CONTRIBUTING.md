# Contributing

This public beta is feedback-first.

## Best ways to help

- Report reproducible product bugs.
- Suggest bounded improvements to usability, accessibility, documentation, or safety.
- Include exact steps, environment details, and sanitized evidence.

## Code contributions

Unsolicited pull requests are not accepted during the controlled public beta unless a maintainer explicitly invites one. This keeps the release boundary reviewable while the architecture settles.

## Security

Do not open public issues for vulnerabilities or sensitive findings. Follow [SECURITY.md](SECURITY.md).
